# Registration Portal Frontend

Frontend application for the Examination Registration Portal.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Configure environment variables (create `.env.local` file):
```
NEXT_PUBLIC_API_BASE_URL=http://localhost:8001
```

3. Run the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:3001`

## Features

- School portal for bulk candidate registration
- Private registration for individual candidates
- Admin portal for managing exams, schedules, and exports
- Timetable viewing and download
